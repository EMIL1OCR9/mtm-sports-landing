#  MTM Sports Landing Page 

Landing page desarrollada para una empresa de construcción de canchas deportivas en México, enfocada en conversión, experiencia de usuario y diseño moderno.

---

##  Características

* Diseño responsive
* UI moderna orientada a conversión
* Arquitectura CSS modular
* Navegación fluida (scroll suave)
* Cotizador interactivo (frontend)
* Galería de proyectos
* Animación en canvas con Three.js

---

##  Tecnologías

* HTML5
* CSS3 (modular: base, layout, components, sections, utilities)
* JavaScript
* Three.js

---

---

##  Objetivo

Desarrollar una landing page funcional enfocada en generar leads reales para un negocio de construcción de canchas deportivas, combinando diseño, desarrollo y estrategia.
